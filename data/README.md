# Daten der Arbeit

Dieser Ordner teilt sich 3 weitere Unterordner auf. Diese beinhalten die verschiedenen angesammelten Daten der einzelnen Analysen. Im Ordner `quantitative_analysis` befinden sich also jegliche Daten der Quantitativen Analyse, welche durch die Skripte im Ordner `evaluation` generiert wurden. Der Ordner `qualitative_analysis` enthält alle ausgefüllten Evaluationsbögen sowie die dazugehörigen Texte. Abschließend enthält der Ordner `study` die Ergebnisse der Verständlichkeitsstudie.
