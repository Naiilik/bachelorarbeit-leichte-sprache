# Auswertung der Ergebnisse

Nachdem alle Daten durch die Durchführung gesammelt wurden, wie in Kapitel [Automatische Text Vereinfachung](../automatic_text_simplification/README.md) beschrieben, kann die Auswertung beginnen.

## Vorraussetzung

- Python 3.14.2

## Verwendung

Wenn alle Daten bereitliegen (die Datei `automatic_text_simplification/generation/results/data.json` muss existieren), kann die Main.py ausgeführt werden. Diese generiert jegliche Statistiken, welche in der Arbeit verwendet werden in den Ordner `evaluation/results` . Die verwendeten Skripte für die einzelnen Plots, befinden sich im Ordner `evaluation/scripts`.
